# TOOLS.md - Notes sur les outils utilisateur (modifiable)

Ce fichier est destiné à _vos_ notes sur les outils externes et les conventions.
Il ne définit pas quels outils existent ; OpenClaw fournit les outils intégrés en interne.

## Exemples

### imsg

- Envoyer un iMessage/SMS : décrivez qui/quoi, confirmez avant d'envoyer.
- Préférez les messages courts ; évitez d'envoyer des secrets.

### sag

- Synthèse vocale : spécifiez la voix, le destinataire/la pièce cible, et si vous souhaitez diffuser en continu.

Ajoutez tout ce que vous voulez que l'assistant sache sur votre chaîne d'outils locale.
