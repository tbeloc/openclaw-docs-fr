# `openclaw setup`

Initialise `~/.openclaw/openclaw.json` et l'espace de travail de l'agent.

Contenu connexe :

- Démarrage rapide : [Démarrage rapide](/start/getting-started)
- Assistant : [Intégration](/start/onboarding)

## Exemples

```bash
openclaw setup
openclaw setup --workspace ~/.openclaw/workspace
```

Exécuter l'assistant via setup :

```bash
openclaw setup --wizard
```
