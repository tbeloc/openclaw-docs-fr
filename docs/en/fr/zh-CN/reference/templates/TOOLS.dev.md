# TOOLS.md - Notes sur les outils utilisateur (modifiable)

Ce fichier est utilisé pour enregistrer *vos* notes sur les outils externes et les conventions.
Il ne définit pas quels outils sont disponibles ; OpenClaw fournit les outils intégrés en interne.

## Exemples

### imsg

- Envoyer iMessage/SMS : décrivez le destinataire/contenu, confirmez avant d'envoyer.
- Essayez d'envoyer des messages courts ; évitez d'envoyer des clés.

### sag

- Synthèse vocale : spécifiez la voix, le haut-parleur/la pièce cible, et si vous utilisez la diffusion en continu.

Ajoutez tout ce que vous souhaitez que l'assistant sache sur votre chaîne d'outils locale.
