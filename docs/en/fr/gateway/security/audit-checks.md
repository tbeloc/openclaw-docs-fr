# Vérifications d'audit de sécurité

`openclaw security audit` émet des résultats structurés identifiés par `checkId`. Cette
page est le catalogue de référence pour ces identifiants. Pour le modèle de menace de haut niveau
et les conseils de durcissement, voir [Sécurité](/fr/gateway/security).

Valeurs `checkId` à signal élevé que vous verrez très probablement dans les déploiements réels (non
exhaustif) :
