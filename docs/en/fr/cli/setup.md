# `openclaw setup`

Initialisez `~/.openclaw/openclaw.json` et l'espace de travail de l'agent.

Liens connexes :

- Démarrage : [Démarrage](/fr/start/getting-started)
- Assistant : [Intégration](/fr/start/onboarding)

## Exemples

```bash
openclaw setup
openclaw setup --workspace ~/.openclaw/workspace
```

Pour exécuter l'assistant via setup :

```bash
openclaw setup --wizard
```
