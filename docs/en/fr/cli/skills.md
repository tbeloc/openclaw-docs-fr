# `openclaw skills`

Inspectez les compétences (intégrées + espace de travail + remplacements gérés) et voyez lesquelles sont éligibles par rapport aux exigences manquantes.

Connexes :

- Système de compétences : [Skills](/tools/skills)
- Configuration des compétences : [Skills config](/tools/skills-config)
- Installations ClawHub : [ClawHub](/tools/clawhub)

## Commandes

```bash
openclaw skills list
openclaw skills list --eligible
openclaw skills info <name>
openclaw skills check
```
