# `openclaw dns`

Assistants DNS pour la découverte étendue (Tailscale + CoreDNS). Actuellement axé sur macOS + Homebrew CoreDNS.

Connexes :

- Découverte de passerelle : [Discovery](/gateway/discovery)
- Configuration de découverte étendue : [Configuration](/gateway/configuration)

## Configuration

```bash
openclaw dns setup
openclaw dns setup --apply
```
