# `openclaw clawbot`

Espace de noms d'alias hérité conservé pour la compatibilité rétroactive.

Alias actuellement supporté :

- `openclaw clawbot qr` (même comportement que [`openclaw qr`](/cli/qr))

## Migration

Préférez les commandes modernes de haut niveau directement :

- `openclaw clawbot qr` -> `openclaw qr`
